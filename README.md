# Library Management System

A Library Management System developed using Java and object-oriented programming (OOP) concepts to manage books, borrowing records, and user information.

## Features
* Add, update, and delete books
* Search books and users
* Borrow and return books
* View borrowing history
* Manage user records

## Technologies Used
* Java
* Object-Oriented Programming (OOP)
* BlueJ

## Project Structure
* Java source files for system logic and GUI
* Text files used for simple data storage

## How to Run
1. Open the project in BlueJ
2. Compile the Java classes
3. Run `MainGUI.java`

## Purpose
This project was created to practice Java programming and apply OOP concepts in building a management system.
